SimplerDevelopment Claude Skills

This installer adds the SimplerDevelopment skills bundle to your Claude installation. After installing, you can draft pages, decks, emails, surveys, and full multi-page websites directly from Claude Desktop or Claude Code.

What gets installed

  • sd-init — bootstrap a project against your tenant
  • sd-create-page, -deck, -email, -survey, -booking-page, -website
  • sd-build-html-embed — single-file HTML embeds
  • sd-learn — per-project feedback capture
  • html-render-block — edit HTML block JSON
  • Design principles + client quickstart docs

Install location

Skills install to ~/.claude/skills/ in your home directory. No administrator password required.

After install

You'll be shown a short configuration snippet to paste into claude_desktop_config.json, then restart Claude Desktop and say "Run sd-init" in any conversation.
